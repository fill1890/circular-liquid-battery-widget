# Options:
Animation mode:  
```
animationMode: [mode number]
```  
1: Wave animation  
2: Pulsing colours  
3: No animation  
  
Position on screen :  
```
topPosition: 100  
leftPosition: 100
```  
Distance in pixels from top-left of screen.
